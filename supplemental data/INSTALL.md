# Tally Time Tracker v1.2.8 — Installation Guide

## Installation

1. Double-click **`TallyTimeTracker_Setup_1.2.8.exe`**
2. If Windows shows a SmartScreen warning, click **More info → Run anyway**
3. Follow the setup wizard and click **Install**
4. Tally will launch automatically when installation completes

The app installs per-user — no administrator password required.

---

## First-Run Setup

1. Tally runs in the **system tray** — look for the icon near the clock in the bottom-right corner
2. Open the **Associations** tab and enter your **Monday.com API token** to enable board integration
3. Tally will begin tracking active windows automatically

To reopen the window at any time, click the tray icon.

---

## Chrome Extension (Optional)

Enables Monday.com task linking from within Chrome.

1. In Chrome, go to `chrome://extensions`
2. Enable **Developer mode** (toggle, top-right)
3. Click **Load unpacked** and select the `browser_extension` folder
4. The extension activates automatically on monday.com pages

---

## Uninstalling

Go to **Settings → Apps**, find *Tally Time Tracker*, and click **Uninstall**.

---

## System Requirements

- Windows 10 or Windows 11
- Chrome (for the browser extension)
- Monday.com account and API token (for board integration)
